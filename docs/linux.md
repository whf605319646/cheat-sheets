<h2>搜索路径</h2>


``` sh

```

<h4>查找 js 或者 jsx 结尾的文件或者目录</h4>

``` sh

```

<h4>只查找文件</h4>
``` sh

```

<h4>查看十分钟内修改的文件</h4>
``` sh
find . -type d -cmin 10
```
